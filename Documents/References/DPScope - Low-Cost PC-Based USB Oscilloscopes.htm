DPScope - Software, Drivers, Documentation
On this page you find everything you need to build, bring up, and use your oscilloscope. Please note the
disclaimer and the copyright notice on the bottom of this page!

  • Assembly and bringup guide with lots of pictures for each step

  • DPScope PC software V1.4.7 for Windows XP (SP3), Vista, 7, 8

  • DPScope PC software with manual connect - use this if your DPScope does not use an FTDI
    based USB-to-serial converter (and thus won't get automatically recognized).

  • Software User Manual and Command Interface Description (the latter will be useful if you want to
    write your own application to control the oscilloscope and get data from it).

  • USB Drivers for Windows 2000, XP, Vista, 7:

    Packed archive (.zip) format
    Executable (.exe) format

  • Installation guide for the USB driver:

    Installation guide for Windows Vista
    Installation guide for Windows 7

    Silkscreen layout (shows component placement)

  • MProg utility (to burn USB configuration file onto the USB converter)

  • MProg User Manual

  • USB Configuration File (use MProg to burn it onto the USB converter cable)
